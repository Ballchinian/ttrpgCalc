This has been a fun project, but far larger than I could imagine it being.
- Ive had to restructure the entire frontend and develop smaller modules while bringing all relevant information through a single object
- So many redesigns on the frontend with fatal flaws with a lack of understanding of what pathfinder wanted. 
- Lots of extra features that Ive felt compelled to add. 

Going forward Ive got lots to improve from and heres a couple tips to future me:
- Fully test outside of programing all the parameters you'll be expected to go to during the real thing. (for example remembering how armour exists ~oops)
- When creating something for the backend, always attach a userID and a uniqueID to it
- Make more functions, seperate them so you can make each one readable with one purpose only
